# ulogix-team.github.io

Repositorio de la página web oficial del equipo Ulogix, publicada en [GitHub Pages](https://ulogix-team.github.io). El sitio documenta el progreso del Proyecto Integrador por módulo del curso, incluye informe de proceso por integrante, entregables y videos.

## Secciones del Sitio

Cada sección corresponde a un módulo del curso e incluye: descripción, video YouTube del entregable, evidencias técnicas y reflexiones del equipo.

| Sección | Contenido |
|---|---|
| **Inicio** | Presentación del equipo, producto y propuesta de valor |
| **Proceso** | VSM, simulación, OEE, análisis de tiempos |
| **Robótica** | Celda robotizada, RobotStudio, análisis de riesgos |
| **Digital Factory** | Gemelo digital, NX, integración I/O virtual |
| **Control** | Grafcet, Ladder, Logix Emulate |
| **SCADA** | Ignition, HMI ISA-101, OPC-UA |
| **Finanzas** | Presupuesto, flujo de caja, VPN, TIR |
| **Informe de Proceso** | Reflexiones individuales y grupales por módulo |
