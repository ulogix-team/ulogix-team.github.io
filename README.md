# ulogix-team.github.io · Sitio Web del Proyecto

Repositorio de la página web oficial del equipo **Ulogix**. Publicado en [GitHub Pages](https://ulogix-team.github.io), el sitio documenta el progreso del Proyecto Integrador del curso APM 2026-1, incluyendo el informe de proceso por módulo, entregables, videos y reflexiones de cada integrante.

</p>

## 🗂️ Secciones del Sitio

Cada sección del sitio corresponde a un módulo del curso y debe incluir:

- Descripción del módulo y objetivos
- Video embebido de YouTube del entregable correspondiente
- Evidencias técnicas (diagramas, pantallas, simulaciones)
- Reflexiones del equipo

| Sección | Contenido |
|---|---|
| **Inicio** | Presentación del equipo, producto y propuesta de valor |
| **Proceso** | Análisis productivo, VSM, simulación, OEE |
| **Robótica** | Celda robotizada, RobotStudio, análisis de riesgos |
| **Digital Factory** | Gemelo digital, NX, integración sensores/actuadores |
| **Control** | Grafcet, Ladder, validación en Logix Emulate |
| **SCADA** | Ignition, HMI ISA-101, OPC, integración gemelo |
| **Finanzas** | Presupuesto, flujo de caja, VPN, TIR |
| **Informe de Proceso** | Reflexiones individuales y grupales por módulo |

## 🚀 Desarrollo Local

```bash
# Clonar el repositorio
git clone https://github.com/ulogix-team/ulogix-team.github.io.git
cd ulogix-team.github.io

# Levantar servidor local (Python)
python3 -m http.server 8000

# Abrir en el navegador
open http://localhost:8000
```
